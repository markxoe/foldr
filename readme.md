<h1 align="center">Foldr</h1>

Switches between file explorer tabs on Windows and MacOS

## Installation

1. Install dependencies:
   `npm i`
2. Build:
   1. for Windows:
      `npm run build-win`
   1. for MacOS:
      `npm run build-mac`

## Development

Install needed dependencies

```bash
  npm i -D
```

Start app

```bash
  npm run start
```
